# 🎉 emoji_loading

A Flutter package to add fun and animated emoji-based loading indicators to your app. Supports bounce, rotate, pulse, and wave animations. 🚀

## ✨ Features
- 🔄 Emoji Rotation
- 📡 Emoji Pulse
- 🌊 Emoji Wave
- 🎉 Emoji Bounce
